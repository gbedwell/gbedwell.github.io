This is an interactive Python script to generate random DNA fragments from the human genome according to different fragmentation strategies. The user is sequentially prompted to enter all of the information needed for fragment generation when the script is run. The intended purpose was to make generating datasets of random fragments a quick and easy process, especially for people unfamiliar with Python.
